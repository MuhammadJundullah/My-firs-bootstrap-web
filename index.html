About Me

Saya adalah mahasiswa semester 4 Teknik Informatika yang memiliki ketertarikan mendalam dalam analisis dan pengolahan data. Saya memiliki kemampuan dalam pengolahan data menggunakan Microsoft Excel, Python, SQL query, dan Tableau untuk visualisasi, serta teknik statistik untuk menggali dan menampilkan wawasan yang berharga dari data. Dengan keahlian ini, saya mampu memahami dan menginterpretasikan data untuk mendukung pengambilan keputusan yang lebih baik. Selain itu, saya juga memiliki kemampuan dalam pembuatan web sederhana dengan menggunakan framework seperti Bootstrap dan Tailwind CSS. Dengan pemahaman yang baik tentang desain responsif dan prinsip-prinsip UX/UI, saya dapat menciptakan tampilan web yang tidak hanya menarik secara visual tetapi juga fungsional dan user-friendly. Saat ini, saya memiliki keinginan kuat untuk mengembangkan kemampuan analisis saya melalui kesempatan magang di lingkungan profesional. Saya yakin bahwa pengalaman praktis ini akan memberikan saya wawasan yang lebih luas dan memperdalam pemahaman saya tentang bagaimana analisis data dapat diterapkan untuk memecahkan masalah nyata di dunia industri.
